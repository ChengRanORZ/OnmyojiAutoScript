配置注意事项:
1. 由于账号匹配时,阴阳师客户端对手机号信息做了处理,所以配置手机账号时,需要注意与阴阳师客户端显示一致
    例如
    - 手机账号为15355554444,
    - 阴阳师客户端显示为153****4444
    - 所以账号配置为153****4444
2. 配置邮箱账号时,直接填写账号即可,不需要额外处理
3. 账号别名(accountAlias) OCR识别正确率低的产物,多个别名使用#分隔
   建议根据日志显示配置
   例子: 
   - 账号:abc0123@163.com
   - 识别结果 a6co123@163.com
      或者 abc023@163.com 等等千奇百怪的结果
   - 别名配置为 a6co123#abc023#.....
   
4. 协作任务查找配置
